📖 高中物理知识体系

高中物理共15章，涵盖力学、电磁学、热学、光学、原子物理等核心知识模块。本知识体系旨在帮助学生建立起完整的物理思维框架，掌握从宏观到微观的分析方法。

点击左侧导航栏可查看各章节详细内容。

第一章：运动的描述与匀变速直线运动

本章概述

本章是整个高中物理的基石，主要引入了描述物体机械运动的基本物理量。

核心概念

  • 质点：理想化模型
  • 参考系：描述物体运动的标准
  • 位移与路程：矢量与标量
  • 速度与速率：描述运动快慢
  • 加速度：速度变化率，连接运动与力的关键

重点公式

v = v₀ + at
x = v₀t + ½at²
v² - v₀² = 2ax

学习目标

建立起用物理语言精确描述物体运动的能力，学会分析简单的运动问题，为后续学习牛顿运动定律做好铺垫。

第二章：相互作用——力

本章概述

本章聚焦于物体运动状态改变的原因——力。

核心内容

  • 力的概念：物体对物体的作用
  • 重力：G = mg
  • 弹力：胡克定律 F = kx
  • 摩擦力：静摩擦力与滑动摩擦力

重点难点

  • 摩擦力的分析（尤其是静摩擦力方向判断）
  • 力的合成与分解（平行四边形定则）
  • 共点力作用下物体的平衡条件

核心方法

受力分析的一般方法：确定研究对象→分析受力→画受力图→列方程求解

第三章：牛顿运动定律

本章概述

这是动力学的核心内容，揭示了力与运动的内在联系。

三大定律

  • 牛顿第一定律（惯性定律）：物体具有惯性，力是改变运动状态的原因
  • 牛顿第二定律：F = ma，定量建立加速度与合外力、质量的关系
  • 牛顿第三定律：作用力与反作用力，大小相等、方向相反

核心公式

F = ma

两类基本问题

  1. 已知受力情况求运动情况
  2. 已知运动情况求受力情况

难点突破

连接体问题、临界极值问题，整体法与隔离法的应用

第四章：曲线运动与万有引力

本章概述

本章将运动从直线拓展到曲线。

曲线运动

  • 速度方向：切线方向
  • 条件：合外力与速度不共线
  • 思想：运动的合成与分解

两种重要模型

  • 平抛运动：水平匀速 + 竖直自由落体
  • 匀速圆周运动：线速度、角速度、周期、向心力

万有引力

F = G·m₁m₂/r²
  • 开普勒三定律
  • 卫星发射与变轨
  • 宇宙速度

第五章：机械能及其守恒定律

本章概述

本章从能量角度审视物理过程。

核心概念

  • 功：做功是能量转化的量度
  • 功率：做功快慢
  • 动能：Eₖ = ½mv²
  • 重力势能：Eₚ = mgh

两大定理

  • 动能定理：合外力做功等于动能变化量
  • 机械能守恒定律：只有重力或弹力做功时，动能与势能相互转化但总量不变

能量观点优势

相比于牛顿定律，能量观点往往能化繁为简，是物理学中一种重要的思维方法。

第六章：静电场

本章概述

本章开始进入电磁学领域。

核心概念

  • 电荷：电荷守恒定律
  • 库仑定律：F = k·q₁q₂/r²
  • 电场强度：E = F/q（力的角度）
  • 电势：φ = E/q（能的角度）

重点内容

  • 带电粒子在电场中的加速和偏转
  • 电容器：电容、充放电、动态分析

思维跨越

从"实物"到"场"的思维跨越是本章的难点。

第七章：恒定电流

本章概述

本章研究电荷在导体中的定向移动。

三个基本物理量

  • 电流：I = q/t
  • 电压：U = IR
  • 电阻：R = ρL/S

核心定律

  • 部分电路欧姆定律：I = U/R
  • 闭合电路欧姆定律：I = E/(R+r)

重点内容

  • 多用电表的改装
  • 滑动变阻器的两种接法（限流式、分压式）
  • 动态电路分析
  • 电源功率和效率问题

第八章：磁场

本章概述

本章研究磁场的性质及其与电流、运动电荷的相互作用。

核心内容

  • 安培力：F = BIL（磁场对通电导线）
  • 洛伦兹力：f = qvB（磁场对运动电荷）

重要应用

  • 质谱仪
  • 回旋加速器
  • 速度选择器

难点

带电粒子在复合场（重力场、电场、磁场共存）中的运动分析

第九章：电磁感应

本章概述

本章揭示了"磁生电"的条件和规律。

核心定律

  • 法拉第电磁感应定律：E = nΔΦ/Δt
  • 楞次定律：增反减同，来拒去留

两类基本模型

  • 动生电动势：E = BLv（导体切割磁感线）
  • 感生电动势：磁场变化产生涡旋电场

难点

电磁感应综合问题：含容电路、导体棒运动分析（单杆、双杆模型）、能量转化问题

第十章：交变电流与传感器

本章概述

本章将电流从直流拓展到交流。

交变电流

  • 正弦式交变电流的产生原理
  • 四个值：瞬时值、峰值、有效值、平均值

变压器

U₁/U₂ = n₁/n₂

基于电磁感应中的互感现象

电能输送

采用高压输电减少线路损耗

传感器

能感知非电学量并将其转化为电学量的器件

第十一章：热学（选修3-3）

本章概述

本章从微观和宏观两个角度研究热现象。

分子动理论

  • 物质由大量分子组成
  • 分子永不停息做无规则热运动
  • 分子间存在相互作用力

理想气体状态方程

pV/T = 恒定

气体实验定律

  • 玻意耳定律：等温变化 pV = 恒定
  • 查理定律：等容变化 p/T = 恒定
  • 盖-吕萨克定律：等压变化 V/T = 恒定

热力学第一定律

ΔU = Q + W

第十二章：机械振动与机械波（选修3-4）

本章概述

本章研究振动及振动在介质中的传播。

机械振动

  • 简谐运动：弹簧振子、单摆
  • 描述物理量：振幅、周期、频率、相位
  • 单摆周期公式：T = 2π√(L/g)

机械波

  • 波的形成机理
  • 横波与纵波
  • 描述物理量：波长、波速、周期 v = λ/T

难点

波的图像及应用、波的干涉、衍射、多普勒效应

第十三章：光（选修3-4）

几何光学

  • 光的折射定律：n = c/v = sinθ₁/sinθ₂
  • 全反射现象和临界角

物理光学

光的本性

第十四章：原子物理与波粒二象性（选修3-5）

本章概述

本章是物理学从经典向量子、微观领域跨越的章节。

波粒二象性

  • 光电效应：Ek = hν - W₀
  • 光的粒子性

原子结构

原子核

  • 衰变规律
  • 核反应方程
  • 质能方程：E = mc²
